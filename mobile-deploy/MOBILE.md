# 모바일 실행

이 프로젝트는 설치형 웹앱(PWA)으로 구성되어 있습니다.

휴대폰 카메라를 사용하려면 사이트를 **HTTPS 주소로 배포**해야 합니다. PC와 같은 Wi-Fi에서 `http://192.168.x.x:8000`으로 접속하는 방식은 모바일 브라우저의 보안 정책 때문에 카메라가 차단됩니다.

HTTPS 배포 후 모바일 Chrome 또는 Safari에서 주소를 열고 카메라 권한을 허용하세요. 브라우저 메뉴의 **홈 화면에 추가**를 선택하면 앱처럼 실행할 수 있습니다. 카메라 화면 오른쪽 위 버튼으로 전면·후면 카메라를 전환할 수 있습니다.

자세 인식 모델은 최초 실행 시 인터넷에서 내려받습니다. 이후 로컬 앱 파일은 서비스 워커에 캐시됩니다.
